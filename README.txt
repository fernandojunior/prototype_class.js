prototype_class.js
------------------
Fernando Felix do Nascimento Junior (http://fjacademic.wordpress.com)

Simple JavaScript Oriented Object Programming Suporter.
Based on Object and Property Descriptors of ECMA5.

Keywords: oriented object programming, javascript, class, prototype, ecma5

Features
========

    - Inheritance
    - Super member (object/prototype and class) access
    - Polymorphism
    - Constructor/initializer
    - Class method concept from python
     
Use
===

Just Load the script prototype_class.js or prototype_class.min.js in your html file. See full examples in exameples folder.
